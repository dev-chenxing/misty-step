# Misty Step

Misty Step is a Morrowind mod built with MWSE-Lua that provides a short-range teleportation spell effect. It adds a "Misty Step" spell to the player and includes an in-game MCM page to configure targeting behavior.

**Key features**

- Adds a `Misty Step` spell that teleports the caster up to ~60 feet (stops before collisions).
- Two targeting modes: `camera` (aim at where you land on the floor) and `facing` (aim in the direction the caster is facing). Configurable via MWSE MCM.
